Dicas de como filtrar o conteúdo da Internet para crianças

Siga o passo a passo e protega seus filhos


Dicas rápidas para o filtrar o Google

  • ✓ Se possível utilizar Google Family Link
  • ✓ No Youtube Ativar "Modo restrito" em Configurações > Conta.
  • ✓ Se possível assinar o YouTube Premium.
  • ✓ Filtrar resultados com conteúdo explícito usando o Google SafeSearch.

Celulares Android

Em aparelhos com Android, é possível restringir os aplicativos, filmes e músicas que podem ser baixados. Para isso, basta entrar na Play Store, ir em Configurações > Família > Controle dos pais. Aqui você pode determinar a faixa etária dos apps e filmes que estão liberados para as crianças.

Controle via DNS

Usar os DNS da Cloudflare 1.1.1.1 para Famílias. Essencial para a filtragem:
1.1.1.3
1.0.0.3
2606:4700:4700::1113
2606:4700:4700::1003

Ou usar os DNS da OpenDNS FamilyShield. Essencial para a filtragem:
208.67.222.123
208.67.220.123

Aplicativos para fixar os servidores DNS para todas as conexões:

  • ✓ No Android - DNS Changer (Configurações > Iniciar Automaticamente)
  • ✓ No iOS - DNSCloak
  • ✓ Em Computadores e Roteadores acesse o guia
